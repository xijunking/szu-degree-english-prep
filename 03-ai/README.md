# AI 辅助学习

这个目录用于沉淀 AI 在备考中的输出内容，让聊天记录变成可复用资料。

## 子目录说明

- `01-prompts`：常用提示词模板
- `02-generated-questions`：AI 生成的题目
- `03-generated-analysis`：AI 生成的解析和讲解
- `04-weakness-drills`：AI 按薄弱点生成的专项强化题
- `11-vocab-grammar`：词汇和语法生成题、知识点讲解
- `12-cloze`：完形填空生成题与逻辑解析
- `13-reading`：阅读文章生成、问题设计、答案定位
- `14-translation`：汉译英句子训练、表达升级
- `15-writing`：作文题、范文、批改反馈
- `05-speaking-writing-feedback`：作文评分、改写和表达优化
- `90-mock-papers`：整套模拟卷、阶段测验和测后分析

## 建议

- 同一主题尽量保留“题目 + 答案 + 解析”三个部分
- 生成内容建议按日期或知识点命名
- 对高质量提示词及时沉淀，后续可重复使用
- 如果某类题型是弱项，优先把相关内容沉淀到对应题型目录，不要混放
