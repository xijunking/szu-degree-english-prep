# 日常练习

这个目录用于记录你每天的练习过程、错题情况和阶段性测试结果。

## 子目录说明

- `01-daily-log`：每日学习记录
- `02-error-log`：错题整理、错因分析、二次复盘
- `03-mock-tests`：模拟测验记录与成绩跟踪
- `11-vocab-grammar`：词汇和语法专项
- `12-cloze`：完形填空专项
- `13-reading`：阅读理解专项
- `14-translation`：汉译英专项
- `15-writing`：短文写作专项
- `90-weekly-review`：周复盘、阶段总结、计划调整

## 建议

- 每天学习后至少留下简短记录
- 错题不要只记答案，要写错因和改进动作
- 模拟测验建议统一记录时间、分数、薄弱项和下一步计划
- 题型专项目录建议一份练习对应一个 Markdown 文件，便于连续积累
- 若某一题型资料较多，可以继续按“真题 / AI题 / 精练题”继续拆分
