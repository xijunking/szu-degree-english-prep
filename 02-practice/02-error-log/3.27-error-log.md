# 3.27 错题记录

## 错题 1

- 来源：2026 深圳大学学位英语样卷练习版
- 题型：词汇和语法
- 题目：The letters PTO _____ Parent-Teacher Organization.
- 选项：
  - A. stand for
  - B. call for
  - C. head for
  - D. care for
- 我的答案：
- 正确答案：A. stand for
- 当前问题：
  - 不认识 `PTO`
  - 知道 `letters` 是“字母”，但不会整句意思
  - 不熟这几个短语的区别
- 生词：
  - `letter` = 字母
  - `letters` = 字母，复数
  - `parent` = 家长；父亲/母亲
  - `teacher` = 老师
  - `organization` = 组织
- 句子意思：
  - 字母 `PTO` 代表 `Parent-Teacher Organization`
  - 可以理解为：`PTO` 是 `Parent-Teacher Organization` 的缩写
- 核心短语：
  - `stand for` = 代表，表示
- 错因分析：
  - 这题主要考“缩写代表什么”的固定表达
  - 看到“字母/缩写 + 完整名称”时，常用 `stand for`
- 短语辨析：
  - `stand for` = 代表，表示
  - `call for` = 需要；要求
  - `head for` = 前往
  - `care for` = 照顾；喜欢
- 例句：
  - `VIP stands for Very Important Person.`
  - `USA stands for the United States of America.`
- 帮助记忆：
  - 缩写“代表”全称，常用 `stand for`

## 今日短语小结

- `stand for` = 代表
- `call for` = 需要；要求
- `head for` = 前往
- `care for` = 照顾；喜欢

## 今日先记住的句子

- `The letters PTO stand for Parent-Teacher Organization.`
- `VIP stands for Very Important Person.`

## 错题 2

- 来源：2026 深圳大学学位英语样卷练习版
- 题型：词汇和语法
- 题目：When applying for a _____ in the office, he was told to see the manager.
- 选项：
  - A. location
  - B. profession
  - C. career
  - D. position
- 我的答案：
- 正确答案：D. position
- 当前问题：
  - 不认识 `told`
  - 不认识 `manager`
  - 选项 B、C、D 不熟
- 生词：
  - `apply for` = 申请
  - `office` = 办公室
  - `told` = 被告诉
  - `manager` = 经理
  - `location` = 地点，位置
  - `profession` = 职业，专业行业
  - `career` = 职业生涯
  - `position` = 职位，岗位
- 句子意思：
  - 当他在办公室申请一个职位时，他被告知去见经理。
- 核心搭配：
  - `apply for a position` = 申请一个职位
  - `see the manager` = 去见经理
  - `be told to do something` = 被告知做某事
- 错因分析：
  - 这题主要考名词词义辨析和常见搭配
  - `position` 最适合和 `apply for` 连用，表示“申请职位”
- 近义词辨析：
  - `location` = 地点
  - `profession` = 职业类别，如 teacher / doctor / lawyer
  - `career` = 一个人长期的职业发展
  - `position` = 某个具体岗位、职位
- 例句：
  - `She applied for a position in the company.`
  - `He wants to be a doctor. It is a good profession.`
  - `He had a successful career.`
- 帮助记忆：
  - 找工作、投简历、申请岗位，常见搭配是 `apply for a position`

## 今日词义辨析补充

- `position`：具体职位、岗位
- `profession`：职业类别
- `career`：职业生涯
- `location`：地点

## 今日再记住两个短语

- `apply for a position`
- `be told to see the manager`

## 错题 3

- 来源：2026 深圳大学学位英语样卷练习版
- 题型：词汇和语法
- 题目：He is _____ that I'd like to introduce him to you.
- 选项：
  - A. such noble person
  - B. so noble person
  - C. such a noble person
  - D. so a noble person
- 我的答案：
- 正确答案：C. such a noble person
- 当前问题：
  - 不认识 `introduce`
  - 不熟 `such` 和 `so` 的用法
  - 选项看起来都差不多
- 生词：
  - `introduce` = 介绍
  - `introduce him to you` = 把他介绍给你
  - `noble` = 高尚的；高贵的
  - `person` = 人
- 句子意思：
  - 他是一个这么高尚的人，以至于我想把他介绍给你。
- 核心句型：
  - `such + a/an + 形容词 + 名词 + that ...`
  - `so + 形容词 + that ...`
- 错因分析：
  - 这题主要考 `such` 和 `so` 的固定结构
  - 当后面是“形容词 + 可数名词单数”时，常用 `such a/an ...`
- 为什么选 C：
  - `person` 是可数名词单数
  - 前面有形容词 `noble`
  - 所以要用 `such a noble person`
- 其他选项为什么不对：
  - A. `such noble person`：少了冠词 `a`
  - B. `so noble person`：`so` 后面不能直接接单数可数名词
  - D. `so a noble person`：结构错误
- 例句：
  - `He is such a kind man that everyone likes him.`
  - `He is so kind that everyone likes him.`
- 帮助记忆：
  - 有“形容词 + 单数名词”时，先想 `such a/an`

## 今日句型补充

- `such + a/an + 形容词 + 名词`
- `so + 形容词`
- `introduce A to B` = 把 A 介绍给 B

## 今日再记住两个句子

- `He is such a noble person that I'd like to introduce him to you.`
- `She is so kind that everyone likes her.`
