## 一、单选题最该复习的点

### 1. 反意疑问句

规则：

- 前面肯定，后面否定
- 前面否定，后面肯定
- `hardly / never / seldom` 也算否定
- 普通动词借 `do / does / did`
- `has + 名词` 表示“有”，常借 `do / does`
- `has + 过去分词` 是完成时，用 `has / hasn't`

必背例句：

```text
He hardly has anything nowadays, does he?
He has a car, doesn't he?
He has finished his homework, hasn't he?
```

### 2. 祈使句

判断方法：

- 动词原形开头
- 没有明确主语
- 像是在叫别人做事
- 常可翻译成“……吧 / 请……”

常见结构：

```text
Have a cigarette, will you?
Open the door, will you?
Sit down, will you?
```

注意区别：

```text
Open the window, will you?
He opens the window every morning, doesn't he?
```

### 3. 数量作整体，用单数

表示时间、距离、金钱、重量时，常把它看成一个整体。

```text
Twenty miles is a long way to cover.
Ten years is a long time.
Five hundred yuan is enough.
```

### 4. 常见固定搭配

这些优先背：

```text
stand for = 代表
apply for a position = 申请职位
be rich in = 富含……
It is no use doing sth = 做某事没有用
be interested in sth = 对某事感兴趣
the key to doing sth = 做某事的关键
used to do sth = 过去常常做某事
get used to doing sth = 习惯于做某事
be keen on doing sth = 热衷于做某事
be eager to do sth = 渴望做某事
prefer to do sth = 更愿意做某事
get through the exams = 通过考试
listen to music = 听音乐
wait for the bus = 等汽车
```

必背例句：

```text
The letters PTO stand for Parent-Teacher Organization.
He applied for a position in the company.
Vegetables are rich in iron.
It is no use pretending.
He is interested in Chinese history.
The key to solving the problem is patience.
He used to drive on the right.
He got used to driving on the left.
Many people are keen on going abroad.
He is eager to go abroad.
```

### 5. such / so

```text
such + a/an + 形容词 + 单数名词 + that ...
so + 形容词 + that ...
```

必背例句：

```text
He is such a noble person that I'd like to introduce him to you.
She is so kind that everyone likes her.
```

### 6. 过去分词作后置定语

看到“被邀请的客人”“被写的书”这类意思，用过去分词。

```text
Some of the guests invited to the party were from other cities.
the book written by him
the school built last year
```

### 7. 倒装

否定意义词放句首时，常用倒装。

```text
Nowhere else in the world can you find more friendly people.
Never have I seen such a thing.
Seldom does he go out.
```

记住：

```text
friendly = 友好的
friend = 朋友
```

### 8. 强调句

结构：

```text
It is/was + 被强调部分 + that + 其他部分
```

必背例句：

```text
It was during the morning rush hour that the accident happened.
It was yesterday that I met him.
```

### 9. 过去虚拟语气

结构：

```text
If + had done, ... would have done
Had + 主语 + done, ... would have done
```

必背例句：

```text
If he had worked harder, he would have got through the exams.
Had he worked harder, he would have got through the exams.
```

### 10. that / those 替代

```text
that = 替代单数名词
those = 替代复数名词
```

必背例句：

```text
The weather in Beijing is colder than that in Shenzhen.
The students in that university are not fewer than those in our university.
```

### 11. required that

结构：

```text
It is required that + 主语 + 动词原形
```

必背例句：

```text
It is required that students attend at least two clubs.
It is necessary that she be careful.
```

## 二、阅读理解考前方法

### 考场方法

1. 先看题目，再回文章找关键词。
2. 不要全文硬翻译。
3. 人名、数字、时间、地点优先定位。
4. 问“why / because / so”时，找因果句。
5. 问“surprised / turning point / infer”时，找文章真正强调的变化点。

### Passage 1 重点

文章主线：

- Gladys 没结婚，也没有孩子
- 她喜欢孩子，经常去儿童医院送泰迪熊
- 她很有钱，最后捐了 1800 万美元给儿童医院
- 她变有钱是因为听老板建议买股票

易错点：

```text
Everyone was surprised that Gladys had so much money.
```

不是惊讶她送礼物，而是惊讶她这么有钱。

```text
She got rich because she did what the boss told her to do.
```

不是老板直接给她钱。

本篇生词：

```text
secretary = 秘书
earned = 挣得
died = 去世
got married = 结婚
daughter = 女儿
brought = 带来
teddy bear = 泰迪熊
lady = 女士
millionaire = 百万富翁
rich = 富有的
surprised = 惊讶的
boss = 老板
stocks = 股票
```

### Passage 2 重点

文章主线：

- 作者小时候说话和阅读都有困难
- 父母一直想帮助他
- 前几年效果不好
- 五年级遇到好老师 Mrs. Wakefield
- 老师帮他建立自信
- 后来作者也成了老师，去鼓励学生

易错点：

```text
amount to much = 有出息；有成就
turning point = 转折点
self-confidence = 自信
```

真正转折点：

```text
In Grade Five.
```

不是三年级转学，而是五年级遇到好老师。

## 三、汉译英保命句

这 5 句直接背：

```text
He bought me a birthday present.
They are waiting for the bus.
I often listen to music.
There have been many changes in my hometown.
The problem is being discussed.
```

对应结构：

```text
buy sb sth = 给某人买某物
wait for = 等待
listen to = 听……
There have been many changes in ... = ……发生了许多变化
is being done = 正在被……
```

考场翻译策略：

- 不会高级句，就写简单句。
- 不要中英混写。
- 先保证主语 + 动词 + 宾语。
- 能写完整句，比乱写高级词更重要。

## 四、作文万能模板

要求：样卷要求至少 100 words。  
建议：写 100-120 词，不要太长。

### 最通用保命版

```text
Nowadays, more and more people pay attention to this topic. It is very important in our daily life.

First, it can help us in many ways. It can make our life easier and better. Second, it can also bring some problems. Some people may spend too much time on it.

I think we should look at it in a right way. We should make good use of it. We should learn more from it. We should not use it in a bad way. Only in this way can we have a better life.
```

### 如果题目是 AI，可以稍微改成这样

```text
Nowadays, more and more people use AI. It is very important in our daily life.

First, we can use AI to study English. It can help us find information quickly and make our study easier. Second, it can also bring some problems. Some people may depend on it too much.

I think AI is useful, but we should use it in a right way. We should make good use of it. Only in this way can we have a better life.
```

### 作文必背词

```text
topic = 话题
daily life = 日常生活
in many ways = 在很多方面
bring some problems = 带来一些问题
spend too much time on it = 在它上面花太多时间
make good use of it = 好好利用它
Only in this way = 只有这样
```

### 作文考场策略

- 一定写三段。
- 不要空作文。
- 句子宁可简单，不要乱写复杂句。
- 不会具体主题时，用 `this topic` 和 `it` 兜底。
- 如果会替换主题词，就把第一句里的 `this topic` 换成题目关键词。
