# 资料整理

这个目录用于集中管理备考资料原件和长期沉淀内容。

## 子目录说明

- `01-syllabus`：考纲、考试须知、报名说明、官方通知
- `02-papers/szu`：深圳大学相关样卷、真题、参考答案
- `02-papers/other`：广东学位英语或其他参考试卷
- `03-vocab`：词汇表、短语表、易混词、核心词汇
- `04-grammar`：语法专题总结、规则归纳、常见陷阱
- `05-writing`：作文模板、开头结尾句、常用表达
- `06-notes`：笔记、方法总结、题型技巧

## 建议

- 官方原始资料优先保留原文件格式
- 同类资料尽量放在一起，不把“笔记”和“原题”混放
- 若后续资料增多，可在子目录内继续按年份或题型细分
